# Sennix - AI Receptionist Demos

Public asset host for AI voice receptionist marketing demos used in Sennix outreach campaigns.

## What is this

Each `*.mp3` is a ~5-10 second AI-generated voiceover showing how a Sennix receptionist might handle an after-hours call for a UK dental practice. The filename is the practice these demos were generated as samples for.

## Want your business name taken down?

If your business is named in a filename and you would like that demo removed, email **privacy@sennix.uk** with the filename. We will remove it within 7 working days. No questions asked.
